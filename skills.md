# Skill: Indeed Scraper & AI Resume Generator

This skill provides an automated pipeline for job discovery and customized resume generation. It is designed to be integrated into agentic frameworks like **OpenClaw** to enable autonomous job application workflows.

## Skill Overview

- **Name**: `indeed_agent_skill`
- **Capabilities**:
  - Market research and job searching on Indeed.com.
  - Full-text job description extraction.
  - AI-driven resume optimization using local LLMs (via Ollama).
  - Consolidated file management for automated pipelines.

## Integration Defaults

| Setting          | Default Value               | Notes                      |
| ---------------- | --------------------------- | -------------------------- |
| Execution Path   | `python3 indeed_scraper.py` | Run from project root      |
| Output Directory | `output/`                   | Consolidated storage       |
| LLM Model        | `llama3.2`                  | Local execution via Ollama |

## Available Commands

### 1. `scrape`

Searches Indeed and extracts job metadata and full descriptions.

**Parameters:**

- `query` (string): Keywords (e.g., "Senior Software Engineer").
- `city` (string): Location (e.g., "Remote", "Austin, TX").
- `days` (integer): Recency limit.
- `pages` (integer): Depth of search.

**Agent Intent Mapping:**

> "Find me Python jobs in New York from the last 3 days."

### 2. `generate`

Customizes a base resume to match a specific job description.

**Parameters:**

- `input` (path): Path to the JSON/CSV file generated by `scrape`.
- `resume` (path): Path to the user's base resume (txt).
- `model` (string): LLM model name.

**Agent Intent Mapping:**

> "Tailor my resume for the jobs we just found in the output folder."

## Usage Patterns for Agents (OpenClaw)

### Autonomous Job Hunter Workflow

1. **Search**: Agent runs `scrape` to populate `output/jobs.csv`.
2. **Analysis**: Agent reads `output/jobs.csv` to filter roles based on high match scores.
3. **Tailoring**: Agent runs `generate` for the top 5 roles.
4. **Finalization**: Agent presents the optimized markdown resumes in `output/` to the user for final approval before submission.

## Prompting Tips for Orchestration

When asking an agent to use this skill, ensure the environment has:

1. `source venv/bin/activate` executed.
2. `ollama serve` running in the background.
3. The base `resume.txt` available in a known directory.
